# EmprenD  🚀

## Sistema Web de Monitoreo y Control de Microemprendimientos Locales Implementando Inteligencia Artificial

> Proyecto académico desarrollado para la carrera de Ingeniería de Sistemas. EmprenD IA es una plataforma web diseñada para apoyar a microemprendimientos locales mediante herramientas de gestión empresarial, monitoreo de inventarios, control financiero y análisis predictivo basado en Inteligencia Artificial.

## 📋 Descripción del Proyecto

EmprenD IA permite a los microemprendedores administrar de manera eficiente sus negocios mediante un sistema web que integra:

* Gestión de productos.
* Control de inventarios.
* Registro de ingresos y egresos.
* Monitoreo de indicadores financieros.
* Predicción de ventas mediante Inteligencia Artificial.
* Alertas automáticas de stock bajo.
* Dashboard de análisis empresarial.
* Captura de prospectos desde la Landing Page.

## 🗄️ Modelo de Datos

La base de datos fue desarrollada en PostgreSQL utilizando Supabase.

### Tablas Implementadas

| Tabla             | Descripción                         |
| ----------------- | ----------------------------------- |
| usuario           | Usuarios registrados                |
| emprendimiento    | Negocios registrados                |
| producto          | Productos del negocio               |
| inventario        | Control de stock                    |
| transaccion       | Registro financiero                 |
| prediccion_ia     | Predicciones generadas              |
| alerta            | Alertas automáticas                 |
| dashboard_kpi     | Indicadores empresariales           |
| prospecto_cliente | Datos capturados en la Landing Page |
| documento         | Documentación del proyecto          |
| reporte_ia        | Reportes generados                  |
| notificacion      | Notificaciones del sistema          |

## 🎯 Objetivo General

Desarrollar un sistema web de monitoreo y control para microemprendimientos locales que permita optimizar la gestión administrativa, financiera y comercial mediante herramientas de Inteligencia Artificial.

---

## 🎯 Objetivos Específicos

* Diseñar una base de datos robusta utilizando PostgreSQL.
* Implementar una Landing Page conectada a Supabase.
* Gestionar inventarios y transacciones.
* Implementar análisis predictivo mediante IA.
* Generar reportes para la toma de decisiones.
* Publicar el sistema en GitHub Pages.
